## Overview
A machine learning project to predict cardiovascular diseases using patient health data.
Built with Python, scikit-learn, and Streamlit.

## Features
- Data preprocessing (cleaning, scaling, encoding)
- Feature selection
- ML Model: Random Forest (90% accuracy, 94% recall)
- Streamlit web app for predictions
